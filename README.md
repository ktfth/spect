# Spect

## Description

A tiny test runner

## Usage

```
[sudo] npm i -g spect
```

After the installation just run:

```
spect ./test/your-test-file.js
```

## Inspiration

The test runner is inspired by tests of [https://github.com/paulirish/source-map-js/blob/cdt/test/run-tests.js](source-map-js).
